# Product Naming Options — v3 (Availability-First)

v1 optimized for conceptual fit. v2 optimized for metaphor range. v3 optimizes for **ownability** — can you actually ship, trademark, and domain-register this name?

## Methodology

1. Ran conflict scans on all v1/v2 shortlist names
2. Generated 20 fresh coined/uncommon names
3. Web-searched each for software conflicts, .com/.app collisions, and trademark risk
4. Filtered to a final 10 that appear clear

## Tone Map

**If the product should feel like a premium tool for serious people:**
→ Ordica, Regnis, Moderis

**If the product should feel like a warm, integrated life system:**
→ Arca, Plenarc, Provisor

**If the product should feel like a modern tech product:**
→ Verteon, Telica, Inflect, Gubernex

## Next Steps Before Committing

1. **Domain check** — run the top 5 through a registrar (Namecheap, Cloudflare) for .app and .com
2. **App Store search** — search macOS App Store and iOS App Store for each name
3. **USPTO / IP Australia search** — check trademark registers in relevant classes (Class 9: software, Class 42: SaaS)
4. **Say it out loud** — "I use [name] to run my life" / "Have you tried [name]?"
5. **Icon test** — does the first letter or a symbol work as an app icon?

## Recommendation

**Ordica** is my top pick. It's:
- Coined (no prior art found)
- 3 syllables, easy to say in any accent
- Instantly suggests "order" without being the word "order"
- Works as app name, company name, and concept
- The "O" makes a strong app icon
- Passes the sentence test: "I track everything in Ordica"

**Arca** is the strongest fallback — a real Latin word, distinctive, 2 syllables, and the most likely to already have domains parked (but worth checking).

Keep `PPP Flow Desktop` through alpha. Pick the final name when you're ready to sign, notarize, and ship to the App Store.
